# curiousTunes

This is a heavily reworked fork of https://github.com/PDavid/aTunes aTunes which I adjust to my own wishes. 
I started out with "rewriting" aTunes, but later understood that it is easier to start from scratch and pull the pieces as I need them.
Let's see what I cam out with

## Features
- Support for huge music collections with thousands of files
- Supports most popular formats: mp3, ogg, wma, wav, flac, mp4...
- Allows tag edition, with advanced features for automatically fill tags properly and batch editing
- Supports several playlists, allowing playlists with thousands of songs
- Allows to organize collection by metadata (artist, album, year, genre, ...) or folder
- <del>Support for external devices to copy or synchronize music</del>
- <del>Supports podcasts</del>
- <del>Supports online radios</del>
- <del>Last.fm scrobbling integrated</del>
- <del>Downloads all information about your music: lyrics, artist information, similar artists, links to Youtube...</del>
- Keep statistics about what you listen to
- <del>Import audio CD to mp3, ogg and include automatically in your collection</del>
- <del>Customizable UI</del>
- Full-screen mode
